## 4. 球评员观点（最难、最关键线，单独成节）

### 方案 A（推荐 · 零合规风险）：LLM 基于实时数据模拟专家观点
- 用决策模型（GLM-5.2/Gemini）基于赛果数据，模拟 3 位风格化球评（Balagué/BBC、Wilson/Guardian、Marcotti/ESPN）的核心观点。
- **学术背书**：StatsBomb TacticalGPT 论文证明，LLM 在真实赛事数据上微调后，What 类问题 50%、Who 类问题 32% 与 ground truth 完全吻合，**事实合理率 96%/94%** —— 证明"用 LLM 基于数据模拟专家级战术观点"技术上可行。⚠️（[论文 PDF](https://blogarchive.statsbomb.com/uploads/2023/10/TacticalGPT-Uncovering-the-Potential-of-LLMs-for-Predicting-Tactical-Decisions-in-Professional-Football.pdf)，投票 1-1 平局，但原文 quote 确凿）
- 报告明确标注"球评员观点为 AI 模拟，不代表其本人立场"（你的 HTML 模板已有此免责声明）。
- **成本**：$0（只用决策模型 token）。
- **风险**：投资人可能觉得"不是真观点"。但你产品定位是"多源数据 + 严密逻辑推导"，模拟观点是逻辑链一环，不是卖点本身。

### 方案 B（真实观点 · 有合规风险）：第三方抓取
- **Apify Twitter scraper**：抓 BBC Sport/ESPN/The Athletic 记者推文，约 **$0.18/1000 条**，**无需信用卡起步**。⚠️（具体单价验证投票 0-1 未通过，但 fetch 原文与搜索一致，以 Apify 官网为准）
- 爬 ESPN/BBC/Sky Sports/The Athletic 专栏：robots.txt/ToS 风险。**Meta v Bright Data (2024) 判例**明确：即使数据公开可访问，违反平台 ToS 抓取仍可能构成违约——合法性取决于合同限制，而非页面是否"公开"。⚠️（[groupbwt](https://groupbwt.com/blog/is-web-scraping-legal/)）
- **成本**：$5-20/月。
- **风险**：ToS 违约、账号封禁、数据不稳定。商用前需法律咨询。

**结论**：MVP/demo 阶段用方案 A（你已在用），商用且投资人强需求真实观点时再评估方案 B 并咨询律师。

## 5. 接入坑与合规汇总

1. **Pinnacle API 已死**（2025-07）—— 别指望免费 sharp odds，走 OddsPapi（含 sharp）或 The Odds API（仅 soft）。
2. **国内支付门槛** —— 国内信用卡海外订阅拒付率 ~90%，WildCard 已停服；用 Dupay/OneKey Card 虚拟卡（USDT 充值）。这是"要不要上付费档"的前置约束。
3. **Transfermarkt ToS 明确禁止爬虫 + 禁止用于训练 AI**（引用 Section 44b UrhG 德国版权法）—— transfermarkt-datasets 是二次爬取产物，商用有违约风险，且周更非实时，不适合分钟级/小时级伤停更新。✅（3-0）
4. **StatsBomb 开放数据禁止商用** —— LICENSE.pdf 原文："The User may not... commercially exploit the data or any analysis derived from the use of the Service"。比"署名"更严，商用需买联盟级许可证。只能用于内部研究/模型训练，不能直接进商用报告。✅（验证虽否决了"署名要求"表述，但挖出更严的"禁商用"条款）
5. **API-Football 免费档 100/天 仅够测试**；Pro $19 是性价比甜点。
6. **X 官方 API 2026 定价**：Free 100 条/月、Basic $200/月、Pro $5000/月。⚠️（你记忆中的"$0.005/条、企业版 $4.2-21 万/月"模型可能有误，需二次确认）
7. **Meta v Bright Data 判例** —— 公开 ≠ 可随便抓，ToS 合同限制是合法性的关键。
